Pengertian HTML

HyperText Markup Language (HTML) merupaka suatu bahasa markah yang digunakan untuk membuat halaman web, menampilkan berbagai informasi di dalam sebuah browser web. HTML dibuat dengan menggunakan aplikasi pengolah kata dan disimpan dalam format ASCII sehingga menjadi halaman web dengan perintah-perintah HTML. HTML merupakan sebuah standar yang diguanakan secara luas untuk menampilkan halaman web. pada saat HTML merupakan standar internet yang didefinisikan dan dikendalikan penggunaannya oleh World Wide Web Consortium (W3C). HTML dibuat oleh Kolaborasi Caillau TIM yang diketuai oleh Berners-lee robert ketika bekerja di CERN pada tahun 1989.
